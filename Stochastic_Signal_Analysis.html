Solved Problems

Problem 1: RC Low-Pass Output Power

Calculate total output power $P_{out}$ given white noise input $N_0/2$.

$$ P_{out} = \int_{-\infty}^{\infty} S_{YY}(f) df = \frac{N_0}{2} \int_{-\infty}^{\infty} \frac{1}{1 + (2\pi f RC)^2} df $$ $$ \text{Result: } P_{out} = \frac{N_0}{4RC} $$
